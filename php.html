Php (Hypertext Preprocessor), or earlier known as personal homepage is an HTML Embedded server side scripting language used in web development. Started in 1994 by Rasmus Lerdorf, php is one of most demanding technology in modern web applications. Php syntax was inherited from C language. Php is open source and easy to learn.

Target audience

  • This course is meant for TOTAL beginners in PHP
  • No programming experience is needed at all
